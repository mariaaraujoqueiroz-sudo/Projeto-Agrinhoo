# 🌱 EcoFuture | Sustentabilidade & Tecnologia

Uma plataforma interativa desenvolvida com foco em sustentabilidade, inovação tecnológica e acessibilidade digital.  
O projeto apresenta soluções inteligentes voltadas ao meio ambiente, utilizando uma interface moderna, responsiva e inclusiva para todos os usuários.

---

# 🎯 Objetivo do Projeto

O principal objetivo deste projeto é demonstrar como a tecnologia pode auxiliar na construção de um futuro mais sustentável.  
A aplicação busca conscientizar os usuários sobre práticas ecológicas, energias renováveis, preservação ambiental e inovação no agronegócio sustentável.

Além disso, o sistema foi pensado para garantir acessibilidade digital, permitindo que diferentes públicos consigam navegar de forma confortável e intuitiva.

---

# 🛠️ Recursos Técnicos

## ♿ Central de Acessibilidade

O projeto conta com um sistema exclusivo de acessibilidade desenvolvido em JavaScript puro.

### Funcionalidades:

- 🔍 **Aumento e redução de fonte**
  - Ajuste dinâmico do tamanho do texto para melhorar a leitura.

- 🌙 **Modo Escuro**
  - Alternância entre tema claro e dark mode para conforto visual.

- 🔊 **Leitura em Voz (Text-to-Speech)**
  - Integração com a Web Speech API para leitura automática das seções.

- 🎨 **Alto Contraste**
  - Melhora a visualização para usuários com baixa visão.

- ⌨️ **Navegação Facilitada**
  - Estrutura preparada para acessibilidade via teclado.

---

# 🎨 UI/UX & Design

O design do projeto foi planejado para transmitir modernidade e sustentabilidade.

### Características Visuais:

- 📱 **Layout Responsivo**
  - Compatível com celulares, tablets e desktops.

- ✨ **Animações Suaves**
  - Utilização de efeitos de scroll animation com Intersection Observer.

- 📰 **Estilo Editorial**
  - Combinação de tipografias modernas para melhor hierarquia visual.

- 🌿 **Paleta Sustentável**
  - Tons verdes, neutros e naturais representando o meio ambiente.

- ⚡ **Performance Otimizada**
  - Estrutura leve e carregamento rápido.
